3 基本参数设定

3.1 外点罚函数基本参数设定
alpha：优化点、设定初始点
sigma：惩罚因子
c：放大因子
epsilon：停机误差
step_max：最大迭代步数

3.2 增广拉格朗日乘子基本参数设定
alpha：优化点、设定初始点
sigma：惩罚因子
rho：放大因子
epsilon：停机误差
yeta：乘子条件误差
stl：乘子条件放大因子
step_max：最大迭代步数
